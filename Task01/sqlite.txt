Файлы необходимые для работы SQLite:
- sqlite3.exe - основной исполняемый файл
- sqlite3.dll - динамическая библиотека
- sqlite3.def - файл определений

Доступные режимы вывода данных в sqlite3:
- ascii - ASCII-текстовые таблицы
- column - выравнивание по колонкам  
- csv - значения, разделенные запятыми
- html - HTML-таблицы
- insert - SQL-операторы INSERT
- json - JSON-формат
- line - каждое значение на отдельной строке
- list - значения через разделитель
- quote - экранирование специальных символов
- tabs - разделение табуляцией
- tcl - TCL-списки
